# Implementation 文档索引

本目录放工程实施和交接资料。它不重新定义产品目标态，也不覆盖机器契约。

| 文档 | 定位 |
|---|---|
| `implementation-handoff.md` | 面向工程师和 AI 的开发任务书、第一版范围和验收标准 |
| `01-implementation-plan.md` | 阶段施工顺序、阶段交付物和验收命令 |
| `api-and-ui-implementation.md` | 已有 API、页面实现落点和前后端对照 |
| `technical-debt-and-refactor-log.md` | 技术债、重构项和后续处理记录 |

新增实现切片时，先确认目标态归属在 `docs/00-system-design.md`、`docs/product/`
或 `docs/backend/`；本目录只记录怎么施工和怎么验收。
