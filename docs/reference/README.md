# Reference 文档索引

本目录放历史事实、样例和参考材料。它们可用于追溯来源，但不能单独覆盖当前目标态、
模块事实源或机器契约。

| 文档 | 定位 |
|---|---|
| `legacy-system-spec.md` | 旧系统关键事实、公司 SQL、旧标签和兼容边界 |
| `data-examples.md` | 数据流、字段和导出样例 |
| `system-blueprint.md` | 历史全量蓝图和页面规格来源，已被当前分层文档拆分治理 |
| `ai-collaboration-engineering-case.md` | 协作工程案例材料 |

当参考材料与当前 `docs/00-system-design.md`、模块事实源或 `config/contracts/*.json`
冲突时，先按当前权威关系修正文档，再进入实现。
